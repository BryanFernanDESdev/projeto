# 🚀 Projeto Entra21

## 📌 Como Executar o Projeto

1. Clone o repositório:

   ```bash
   git clone https://github.com/seu-usuario/projeto-entra21.git
   ```

2. Acesse a pasta do projeto:

   ```bash
   cd projeto-entra21
   ```

3. Abra o arquivo `index.html` em seu navegador.
